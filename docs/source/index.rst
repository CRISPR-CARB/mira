MIRA Documentation
==================
MIRA is a framework for representing systems using ontology-grounded
**meta-model templates**, and generating various model implementations and
exchange formats from these templates. It also implements algorithms for
assembling and querying **domain knowledge graphs** in support of modeling.

Installation
------------

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    $ pip install git+https://github.com/indralab/mira.git

To install in development mode, use the following:

.. code-block:: shell

    $ git clone git+https://github.com/indralab/mira.git
    $ cd mira
    $ pip install -e .
